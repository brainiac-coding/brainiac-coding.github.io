Structure Rules

HTML is made up of Tags. Tags contain text. Depending on the tag, the text will show up in a certain way, show an image or shape, or show a link or button. The text goes inbetween an anding and starting tag. The starting tag looks like this:<tag name>. The ending tag looks like this: </tag name>. Tags can contain attributes. Attributes are declared in the starting tag. Here's an example: <Tag Name Attribute="...">.Tags that you want to show on your website go in the body tag (already on the Code Editor). Tags that go inside of the head tag don't show up on your website. They just tell your website how to work. For example, the style tag goes in the head. CSS coding goes in the style tag. CSS follows this format:

tag name {		
		property: value;
		property: value;
		property: value;
		...
}

You can style as many tags as you want in the style tag. In CSS, the tag name is known as the selector because it selects what tag you are styling. Each row inbetween the curly brackets {} is called a declaration because it declares the value of a property.

Here is a list of tag names with descriptions:

  • p The p is short for paragraph. Whatever you type inbetween the starting and ending tags shows up as small, thin text. Ex. <p>This will make a paragraph.</p>
  • h(1-6) The h is short for heading. h1 is the largest and thickest, and h6 is the smallest and thinnest. Ex. <h1>This will make a heading.</h1>
  • a The a tag is for hyperlinks. The link address is put in the "href" attribute. Ex. <a href="linkAddress">Click Me!</a>
  • img "img" is short for image. The img tag is special because rather than an ending tag, it has a slash after the tag name and attribute(s). The image address is put in the "src" attribute. Ex. <img src="imgAddress"/>.
  • ul "ul" is short for un-ordered list (bullet points). The li tag (list) is used for each bullet point. Ex. <ul><li>Bullet Point</li><li>Bullet Point</li><li>Bullet Point</li></ul>
  • ol "ol" is short for ordered list (numbered list). It also uses the li tag. Ex. <ol><li>1</li><li>2</li><li>3</li></ol>
